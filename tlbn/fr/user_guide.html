1. Premiers pas

  1. Installez l'extension depuis le Chrome Web Store.
  2. Cliquez sur l'icône de l'extension dans la barre Chrome pour ouvrir le panneau latéral.
  3. Utilisez les onglets en haut :
    • Tableaux — liste des tableaux et actions rapides
    • Import/Export — échange de fichiers OBZ/OBF
    • Paramètres — valeurs par défaut de synthèse vocale

Astuce : cliquez sur ? dans l'en-tête à tout moment pour rouvrir ce guide.

2. Créer un nouveau tableau

  1. Ouvrez l'onglet Tableaux.
  2. Cliquez sur le bouton + dans l'en-tête.
  3. Saisissez le nom du tableau et cliquez sur Enregistrer.
  4. Le nouveau tableau s'ouvre dans un onglet plein du navigateur en mode Paramètres.

3. Taille du tableau et grille

Dans l'onglet du tableau, cliquez sur ⚙ Paramètres pour modifier les options de toute la grille :

  • Lignes et Colonnes — taille de la grille
  • Police — taille du texte sur les boutons
  • Carré — boutons carrés

Les modifications sont enregistrées immédiatement.

Quand vous avez terminé, cliquez sur ✓ Terminé pour quitter le mode Paramètres et revenir à l'utilisation normale du tableau (mode parole).

4. Personnaliser les boutons

Dans l'onglet du tableau, en mode Paramètres, cliquez sur un bouton pour le modifier :

4.1 Texte du bouton

  • Définissez le texte principal — il est affiché et prononcé.
  • Ajoutez si besoin une note ou un sous-titre.

4.2 Langue du bouton

  • Chaque bouton peut utiliser la langue et la voix par défaut.
  • Ou vous pouvez définir une langue spécifique pour ce bouton.

4.3 Arrière-plan du bouton

  • Choisissez la couleur de fond.
  • Ajoutez ou retirez l'image de fond.

4.4 Audio personnalisé (remplace la synthèse vocale)

Chaque bouton peut parler avec le moteur de synthèse vocale (TTS) du navigateur ou avec un message enregistré. Si un enregistrement est présent, il est prioritaire : en appuyant sur le bouton, c'est votre audio qui est lu, pas la voix synthétisée.

  • Utilisez Enregistrer, Lire et Arrêter pour enregistrer et écouter le message.
  • Microphone : la première fois, Chrome peut demander l'accès au microphone. Autorisez-le pour ce site.
  • Préécoute : utilisez Lire pour écouter avant de fermer l'éditeur.
  • Retour au TTS : appuyez sur Supprimer à côté de l'enregistrement pour le supprimer et réutiliser la synthèse pour ce bouton.
  • OBZ/OBF : les tableaux avec audio personnalisé s'importent et s'exportent comme les autres ; si le format le permet, l'enregistrement reste lié au bouton.

Rappel : l'audio personnalisé remplace la synthèse seulement pour ce bouton. La langue et la voix par défaut du panneau latéral s'appliquent encore aux boutons sans enregistrement ni langue propre.

4.5 Effacer, Annuler et Enregistrer

En bas de l'éditeur de bouton :

  • Effacer — vide le bouton (supprime texte, note, langue, couleurs, image de fond et audio). Enregistrement immédiat et fermeture de l'éditeur.
  • Annuler — ferme l'éditeur sans appliquer les changements de cette session. Le bouton reste comme la dernière sauvegarde.
  • Enregistrer — enregistre tout le contenu de l'éditeur (y compris fond et audio) et ferme l'éditeur.

Vous pouvez aussi appuyer sur Échap — comme Annuler.

5. Ordre des boutons

En mode Paramètres, faites glisser les boutons pour changer leur position.

Le nouvel ordre est enregistré automatiquement.

6. Langue et voix par défaut

  1. Ouvrez l'onglet Paramètres du panneau latéral.
  2. Sélectionnez Langue par défaut.
  3. Choisissez la Voix pour cette langue.
  4. Cliquez sur ▶ Tester pour entendre un extrait.

Les voix enregistrées sont mémorisées par langue et utilisées lorsqu'un bouton n'a pas sa propre langue ou voix.

7. Import, export et sauvegarde (OBZ/OBF)

7.1 Importer des tableaux

  1. Ouvrez l'onglet Import/Export.
  2. Choisissez le mode d'importation :
    • Fusionner — ajouter aux tableaux existants
    • Remplacer — tout supprimer, puis importer
  3. Choisissez un fichier .obz ou .obf.

7.2 Exporter des tableaux

  • ⬇ Exporter tous les tableaux crée un fichier de sauvegarde .obz.
  • Dans la liste des tableaux, utilisez l'action d'export sur la carte d'un tableau pour n'exporter que celui-ci.

7.3 Partage multiplateforme

Les tableaux sont compatibles avec l'application Talking Buttons pour Android via Open Board Format. Vous pouvez exporter depuis l'extension et importer sur Android, ou l'inverse.

8. Conseils et différences avec le mobile

  • L'extension Chrome ouvre les tableaux dans un onglet normal du navigateur pour parler ou modifier.
  • Les voix TTS dépendent des voix installées dans le navigateur ou le système.
