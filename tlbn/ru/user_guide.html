«Говорящие кнопки» — руководство пользователя

Как создавать АДК-доски, настраивать кнопки, выбирать голоса и обмениваться досками в формате Open Board Format.

1. Первые шаги

  1. Установите расширение из интернет-магазина Chrome.
  2. Нажмите значок расширения на панели Chrome, чтобы открыть боковую панель.
  3. Вверху доступны вкладки:
    • Доски — список досок и быстрые действия
    • Импорт/Экспорт — обмен файлами OBZ/OBF
    • Настройки — параметры синтеза речи по умолчанию

Совет: в любой момент нажмите ? в заголовке, чтобы снова открыть это руководство.

2. Новая доска

  1. Откройте вкладку Доски.
  2. Нажмите кнопку + в заголовке.
  3. Введите название доски и нажмите Сохранить.
  4. Новая доска откроется на всю вкладку браузера в режиме настроек.

3. Размер доски и сетка

На вкладке доски нажмите ⚙ Настройки, чтобы изменить параметры сетки кнопок:

  • Строк и Столбцов — размер сетки
  • Шрифт — размер текста на кнопках
  • Квадрат — квадратные кнопки

Изменения сохраняются сразу.

Когда закончите, нажмите ✓ Готово, чтобы выйти из режима настроек и вернуться к обычному использованию доски (режим речи).

4. Настройка кнопок

На вкладке доски в режиме настроек нажмите любую кнопку, чтобы открыть окно настроек:

4.1 Текст кнопки

  • Задайте основной текст — он показывается и произносится.
  • При необходимости добавьте заметку (подпись).

4.2 Язык кнопки

  • Для кнопки можно использовать язык и голос по умолчанию.
  • Или выбрать отдельный язык для этой кнопки.

4.3 Фон кнопки

  • Выберите цвет фона.
  • Добавьте или удалите фоновое изображение.

4.4 Своё аудио (вместо синтеза речи)

Каждая кнопка может говорить с помощью встроенного в браузер синтеза речи (TTS) или записанного вами сообщения. Пока есть запись, она имеет приоритет: нажатие на кнопку воспроизводит вашу запись, а не синтезированную речь.

  • Используйте Запись, Воспр. и Стоп, чтобы записать и прослушать сообщение.
  • Микрофон: при первой записи Chrome может запросить доступ к микрофону. Разрешите его для этого сайта.
  • Прослушивание записи: нажмите Воспр., чтобы проверить запись перед закрытием редактора.
  • Вернуться к синтезу речи: нажмите Удалить рядом с записью, чтобы убрать её и снова использовать TTS для этой кнопки.
  • OBZ/OBF: доски со своим аудио можно импортировать и экспортировать как обычно; при поддержке формата, запись сохраняется вместе с кнопкой.

Имейте в виду: аудио заменяет синтез только для этой кнопки. Язык и голос по умолчанию в боковой панели по-прежнему действуют для кнопок без записи и без своего языка.

4.5 Очистить, Отмена и Сохранить

Кнопки внизу окна настроек:

  • Очистить — сбрасывает кнопку в пустую ячейку (убирает текст, заметку, язык, цвета, фон и запись). Сохраняется сразу, окно настроек закрывается.
  • Отмена — закрывает окно настроек без сохранения изменений этой сессии. На кнопке остаётся последнее сохранённое состояние.
  • Сохранить — сохраняет всё (в том числе фон и запись) и закрывает окно настроек.

Также можно нажать Esc — эффект как у Отмена.

5. Порядок кнопок

В режиме настроек на вкладке доски перетаскивайте кнопки, меняя их порядок.

Новый порядок сохраняется автоматически.

6. Язык и голос по умолчанию

  1. Откройте вкладку Настройки в боковой панели.
  2. Выберите Язык по умолчанию.
  3. Выберите Голос для этого языка.
  4. Нажмите ▶ Тест, чтобы прослушать пример.

Выбранные голоса запоминаются для каждого языка и используются, если у кнопки не задан другой язык или голос.

7. Импорт, экспорт и резервная копия (OBZ/OBF)

7.1 Импорт досок

  1. Откройте вкладку Импорт/Экспорт.
  2. Выберите режим импорта:
    • Добавить — к существующим доскам
    • Заменить — удалить все, затем импортировать
  3. Выберите файл .obz или .obf.

7.2 Экспорт досок

  • ⬇ Экспортировать все доски создаёт один файл резервной копии .obz.
  • В списке досок используйте кнопку экспорта на карточке доски, чтобы выгрузить одну доску.

7.3 Обмен между устройствами

Доски совместимы с приложением «Говорящие кнопки» для Android через Open Board Format. Можно экспортировать из расширения и импортировать в Android или наоборот.

8. Советы и отличия от мобильного приложения

  • Расширение открывает доски в обычной вкладке браузера для прослушивания и редактирования.
  • Набор голосов синтеза речи зависит от браузера и системы.
