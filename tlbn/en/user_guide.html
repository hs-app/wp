1. Getting Started

  1. Install the extension from Chrome Web Store.
  2. Click the extension icon in Chrome toolbar to open the side panel.
  3. Use tabs at the top:
    • Boards - board list and quick actions
    • Import/Export - OBZ/OBF file exchange
    • Settings - Text-to-Speech defaults

Tip: Click ? in the header any time to reopen this guide.

2. Create a New Board

  1. Open Boards tab.
  2. Click the + button in the header.
  3. Enter board name and click Save.
  4. The new board opens in a full browser tab in Settings mode.

3. Change Board Size and Grid Options

In board tab, click Settings to edit layout-wide options:

  • Rows and Cols for grid size
  • Font for button text size
  • Square toggle for square buttons

Changes are saved immediately.

Click ✓ Done when you are finished to exit Settings mode and return to normal board use (speaking mode).

4. Customize Buttons

In board tab and Settings mode, click any button to edit:

4.1 Button Text

  • Set the main text to be shown and spoken.
  • Add optional note/subtitle text.

4.2 Button Language

  • Each button can use Default language/voice.
  • Or set a specific language override for that button.

4.3 Button Background

  • Choose background color.
  • Choose/remove background image.

4.4 Custom audio (overrides TTS)

Each button can speak using the browser's Text-to-Speech (TTS) engine, or use a custom recorded message instead. When a recording is present, it always takes priority: tapping the button plays your audio, not synthesized speech.

  • Use Record, Play, and Stop to capture and preview a message.
  • Microphone: The first time you record, Chrome may ask for microphone permission. Allow it for this site so recording works.
  • Preview: Use Play to listen before leaving the editor.
  • Back to TTS: Tap Remove next to the recording to delete it and use TTS again for that button.
  • OBZ/OBF: Boards that include custom audio can be imported and exported like other boards; the recording travels with the button when supported.

Remember: Custom audio overrides TTS only for that button. Default language and voice in the side panel still apply to buttons without a recording or language override.

4.5 Clear, Cancel, and Save

At the bottom of the button editor:

  • Clear — Resets this button to an empty slot (removes text, note, language override, colors, background image, and custom audio). This is saved right away and the settings window closes.
  • Cancel — Closes the settings window without applying changes from this edit session. The button stays as it was last saved.
  • Save — Stores everything in the editor (including background image and custom audio) and closes the settings window.

You can also press Esc to close the editor the same way as Cancel.

5. Reorder Buttons

In board Settings mode, drag and drop buttons to change their position.

The new order is saved automatically.

6. Set Default Language and Voice

  1. Go to side panel Settings tab.
  2. Select Default language.
  3. Select preferred Voice for that language.
  4. Click Test to preview speech.

Saved voices are remembered per language and used when a button does not override language/voice.

7. Import, Export, and Backup (OBZ/OBF)

7.1 Import Boards

  1. Open Import/Export tab.
  2. Select import mode:
    • Merge - add imported boards to existing ones
    • Replace - delete existing boards, then import
  3. Choose a .obz or .obf file.

7.2 Export Boards

  • Export all boards creates one .obz backup file.
  • From board list, use export action on a board card to export a single board.

7.3 Cross-platform Sharing

Boards are compatible with the Android Talking Buttons app through Open Board Format. You can export from extension and import into Android, or the opposite.

8. Tips and Known Differences from Mobile

  • Chrome extension opens boards in a regular browser tab for speaking/editing.
  • TTS voices depend on browser/OS installed voices.
