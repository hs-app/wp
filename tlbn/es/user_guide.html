1. Primeros pasos

  1. Instale la extensión desde Chrome Web Store.
  2. Haga clic en el icono de la extensión en la barra de Chrome para abrir el panel lateral.
  3. Use las pestañas superiores:
    • Tableros — lista de tableros y acciones rápidas
    • Importar/Exportar — intercambio de archivos OBZ/OBF
    • Configuración — valores predeterminados de síntesis de voz

Consejo: pulse ? en el encabezado en cualquier momento para volver a abrir esta guía.

2. Crear un tablero nuevo

  1. Abra la pestaña Tableros.
  2. Pulse el botón + del encabezado.
  3. Escriba el nombre del tablero y pulse Guardar.
  4. El tablero nuevo se abre en una pestaña completa del navegador en modo Configuración.

3. Tamaño del tablero y cuadrícula

En la pestaña del tablero, pulse ⚙ Configuración para editar las opciones de toda la cuadrícula:

  • Filas y Columnas — tamaño de la cuadrícula
  • Fuente — tamaño del texto del botón
  • Cuadrado — botones cuadrados

Los cambios se guardan al instante.

Cuando termine, pulse ✓ Listo para salir del modo Configuración y volver al uso normal del tablero (modo de habla).

4. Personalizar botones

En la pestaña del tablero, en modo Configuración, pulse cualquier botón para editarlo:

4.1 Texto del botón

  • Defina el texto principal que se muestra y se pronuncia.
  • Opcionalmente añada una nota o subtítulo.

4.2 Idioma del botón

  • Cada botón puede usar el idioma y la voz predeterminados.
  • O bien puede fijar un idioma concreto para ese botón.

4.3 Fondo del botón

  • Elija el color de fondo.
  • Añada o quite la imagen de fondo.

4.4 Audio personalizado (reemplaza la síntesis de voz)

Cada botón puede hablar con el motor de síntesis de voz (TTS) del navegador o con un mensaje grabado. Si hay una grabación, tiene prioridad: al pulsar el botón se reproduce su audio, no la voz sintetizada.

  • Use Grabar, Reproducir y Parar para grabar y escuchar el mensaje.
  • Micrófono: la primera vez que graba, Chrome puede pedir permiso de micrófono. Permítalo para este sitio.
  • Preescucha: use Reproducir para escuchar antes de cerrar el editor.
  • Volver a TTS: pulse Eliminar junto a la grabación para borrarla y usar de nuevo la síntesis para ese botón.
  • OBZ/OBF: los tableros con audio personalizado se pueden importar y exportar como los demás; si el formato lo admite, la grabación va con el botón.

Recuerde: el audio personalizado solo sustituye la síntesis en ese botón. El idioma y la voz predeterminados del panel lateral siguen aplicándose a los botones sin grabación ni idioma propio.

4.5 Borrar, Cancelar y Guardar

En la parte inferior del editor de botones:

  • Borrar — deja el botón vacío (quita texto, nota, idioma, colores, imagen de fondo y audio). Se guarda al instante y se cierra el editor.
  • Cancelar — cierra el editor sin guardar los cambios de esta sesión. El botón queda como última vez guardado.
  • Guardar — guarda todo lo del editor (incluidos fondo y audio) y cierra el editor.

También puede pulsar Esc para cerrar como con Cancelar.

5. Orden de los botones

En el modo Configuración, arrastre y suelte los botones para cambiar su posición.

El nuevo orden se guarda automáticamente.

6. Idioma y voz predeterminados

  1. Vaya a la pestaña Configuración del panel lateral.
  2. Seleccione Idioma predeterminado.
  3. Elija la Voz para ese idioma.
  4. Pulse ▶ Probar para escuchar una muestra.

Las voces guardadas se recuerdan por idioma y se usan cuando a un botón no se le asigna otro idioma/voz.

7. Importar, exportar y copia de seguridad (OBZ/OBF)

7.1 Importar tableros

  1. Abra la pestaña Importar/Exportar.
  2. Elija el modo de importación:
    • Fusionar — añadir a los tableros existentes
    • Reemplazar — eliminar todos, luego importar
  3. Elija un archivo .obz o .obf.

7.2 Exportar tableros

  • ⬇ Exportar todos los tableros crea un archivo de copia de seguridad .obz.
  • En la lista de tableros, use la acción de exportar en la tarjeta de un tablero para exportar solo ese tablero.

7.3 Uso entre dispositivos

Los tableros son compatibles con la aplicación Talking Buttons para Android mediante Open Board Format. Puede exportar desde la extensión e importar en Android, o al revés.

8. Consejos y diferencias respecto al móvil

  • La extensión de Chrome abre los tableros en una pestaña normal del navegador para hablar o editar.
  • Las voces de síntesis dependen de las voces instaladas en el navegador o el sistema.
